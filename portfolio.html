Python course

This course covers the foundational concepts of Python programming: variables, expressions, statements, functions, loops, strings, and lists.

This course is designed for high school students who are interested in learning computer science and python. The first program is a calculator, then a class attendance counter, and a password validation program.

Each of these programs is designed to run in the CLI. Each program comes with 2-3 versions with varying difficulty. Program outline, instructions, solutions and unit testing are included in the course materials.

Calculator | Counter | Password Validator
